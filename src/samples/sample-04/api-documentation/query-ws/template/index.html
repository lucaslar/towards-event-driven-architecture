    • #1 Example - Dorothy Hodgkin

      Search with all paramters.

      Laureates born in 1995 or later (i.e. after 1994)

      {

      "operator": "gt",
      "year": 1994
    }
  },
  "clientRef": "Some String"
}
    • #3 Example - Alber eins (no client ref)

      Search with partial strings.

      {
  "queryData": {
    "firstName": "Alber",
    "lastName": "eins"
  }
}
    • #4 Example - Berliners

      All laureates born and passed in 'u33' (geo hash)

      {
  "queryData": {
    "birthLocation": [
      "u33"
    ],
    "deathLocation": [
      "u33"
    ]
  },
  "clientRef": "Some String"
}
